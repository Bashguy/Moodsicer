How It Works

Convolutional Neural Network (CNN)

Our emotion detection system uses a sophisticated CNN trained on facial expressions. It processes images through multiple layers to identify facial features associated with different emotions.

Real-Time Processing

1. Your webcam captures video frames

2. Each frame is sent through our CNN

3. Facial features are analyzed

4. The model predicts emotion with confidence

5. Results appear instantly

CNN Architecture

Input Layer
Receives 48x48 grayscale images

Convolutional Layers
Extract facial features using filters

Pooling Layers
Reduce spatial dimensions

Fully Connected Layers
Process extracted features

Dropout Layers
Prevent overfitting

Output Layer
Predicts emotion probabilities

Accuracy

Each prediction includes a confidence score, letting you see how reliable the detection is.
